abstract
In Java abstract keyword can be used with classes and methods.
Abstract class: is a restricted class that cannot be used to create objects (to access it, it must be inherited from another class).
Abstract method: can only be used in an abstract class, and it does not have a body. The body is provided by the subclass (inherited from).
boolean
boolean keyword is used to declare a variable as a boolean type, which represents only either true or false.
break
break keyword is used to terminate for, while, or do-while loop; it may also be used to terminate a switch statement as well.
byte
byte keyword is used to declare a variable as a numeric type; can hold an 8-bit integer number which ranges from -128 to 127.
case
case keyword is used with switch keyword in Java to specify a code block which needs to be executed in case of a specific value is passed in switch statement; it is followed by the value to be compared to and a colon.
catch
catch is used in conjunction with a try block (and an optional finally block); the statements in the catch block specify what to do if a specific type of exception is thrown by the try block.
